## 基于范围的for循环
C++11 新增了一种循环：基于范围的`for`循环，下面几种简单用法
```cpp
double prrices[5] = {4.99, 10.99, 6.87, 7.99, 89.4};
for (double x : prices)
    std::cout << x << " ";
    std::cout << std::endl;
// 这里的&表示引用变量，这种声明能让接下来的代码直接修改数组的内容
for (double & x : prices)
    x = x * 0.8; 
for (int x : {3, 5, 2, 8, 6})
    std::cout << x << " ";
    std::cout << std::endl;
```
